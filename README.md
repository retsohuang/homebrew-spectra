# homebrew-spectra

Homebrew tap for [Spectra](https://github.com/kaochenlong/spectra-app).

## Installation

```sh
brew tap retsohuang/spectra
brew install --cask spectra
```

## Updates

A daily GitHub Actions workflow automatically checks for new Spectra releases and updates the cask formula.

To check manually:

```sh
brew livecheck spectra
brew upgrade --cask spectra
```
